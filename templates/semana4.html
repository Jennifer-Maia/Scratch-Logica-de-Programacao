Semana 4: Trabalhando com Variáveis

Objetivo

Ensinar os alunos a criar e usar variáveis no Scratch para armazenar e manipular dados, através de atividades práticas como a criação de um jogo de pontos.

Atividades

1. Introdução às Variáveis (20 minutos)

  1. Explicação do que são variáveis e sua importância na programação.
  2. Discussão sobre os tipos de dados que podem ser armazenados em variáveis (números, texto).
  3. Exemplos do mundo real de variáveis (ex.: pontuação em jogos, saldo bancário).

2. Criação e Uso de Variáveis no Scratch (30 minutos)

  1. Demonstrar como criar uma variável no Scratch:
    1. Clicar em "Variáveis" no painel de blocos.
    2. Clicar em "Criar uma variável" e dar um nome significativo (ex.: "pontuação").
  2. Mostrar como definir e alterar o valor de uma variável:
    1. Adicionar o bloco "definir [variável] para [valor]" no script.
    2. Adicionar o bloco "mudar [variável] por [valor]" no script.
  3. Exemplo Prático:
    • Criar uma variável "pontuação" que aumenta quando o sprite toca em um objeto específico.
    • Passo a passo:
      1. Adicionar um sprite para o personagem e vários sprites para os objetos.
      2. Adicionar um bloco "quando a bandeira verde for clicada".
      3. Adicionar um bloco "definir [pontuação] para 0".
      4. Adicionar um bloco "para sempre" que contém uma condição "se tocando em [objeto] então".
      5. Dentro da condição, adicionar um bloco "mudar [pontuação] por 1".

3. Jogo de Pontos (40 minutos)

  1. Desenvolver um jogo simples onde os alunos aplicam o uso de variáveis para controlar a pontuação.
  2. Passo a passo:
    1. Adicionar um sprite para o jogador e vários sprites para os objetos a serem coletados.
    2. Adicionar uma variável "pontuação".
    3. Configurar o jogo para iniciar a pontuação em 0.
    4. Adicionar lógica para aumentar a pontuação quando o jogador coleta um objeto.
    5. Adicionar lógica para finalizar o jogo quando a pontuação atinge um valor específico.

4. Discussão e Feedback (30 minutos)

  1. Discutir as dificuldades encontradas pelos alunos durante a atividade.
  2. Compartilhar diferentes abordagens e soluções para os problemas.
  3. Feedback sobre o que os alunos acharam da atividade e do uso de variáveis.

Materiais Necessários

  • Computadores com acesso à internet e ao site do Scratch.
  • Projetor para demonstração.

Duração Total

2 horas
