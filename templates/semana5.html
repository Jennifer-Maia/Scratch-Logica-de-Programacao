Semana 5: Eventos e Interações

Objetivo

Ensinar os alunos a usar eventos para interações entre personagens e criar animações interativas no Scratch.

Atividades

1. Introdução aos Eventos (20 minutos)

  1. Explicação do conceito de eventos em programação e sua importância para interações dinâmicas.
  2. Exemplos de eventos no mundo real (ex.: pressionar um botão, clicar em um link).

2. Eventos Básicos no Scratch (30 minutos)

  1. Demonstrar diferentes tipos de eventos no Scratch:
    • Quando a bandeira verde é clicada.
    • Quando uma tecla específica é pressionada.
    • Quando um sprite é clicado.
    • Transmissão e recepção de mensagens.
  2. Exemplo Prático:
    • Criar um sprite que se move quando a tecla de seta para a direita é pressionada.
    • Passo a passo:
      1. Adicionar um sprite.
      2. Adicionar um bloco "quando a bandeira verde for clicada".
      3. Adicionar um bloco "para sempre".
      4. Dentro do loop, adicionar um bloco "se tecla [seta para a direita] pressionada então".
      5. Dentro da condição, adicionar o bloco "mova 10 passos".

3. Comunicação entre Personagens (40 minutos)

  1. Demonstrar como os sprites podem se comunicar entre si utilizando mensagens.
  2. Exemplo Prático:
    • Criar dois sprites, onde um envia uma mensagem e o outro reage a ela.
    • Passo a passo:
      1. Adicionar dois sprites.
      2. No primeiro sprite, adicionar um bloco "quando este sprite for clicado".
      3. Adicionar um bloco "transmita [mensagem1]".
      4. No segundo sprite, adicionar um bloco "quando eu receber [mensagem1]".
      5. Adicionar um bloco "diga Olá por 2 segundos".
  3. Exemplo Prático 2:
    • Criar uma animação onde um sprite pergunta algo e outro responde.
    • Passo a passo:
      1. Adicionar dois sprites.
      2. No primeiro sprite, adicionar um bloco "quando a bandeira verde for clicada".
      3. Adicionar um bloco "diga Olá, como você está? por 2 segundos".
      4. Adicionar um bloco "transmita [mensagem2]".
      5. No segundo sprite, adicionar um bloco "quando eu receber [mensagem2]".
      6. Adicionar um bloco "diga Estou bem, obrigado! por 2 segundos".

4. Exercícios Práticos (30 minutos)

  1. Exercício 1: Criação de Jogo Interativo
    • Desenvolver um jogo onde os personagens interagem através de eventos.
    • Passo a passo:
      1. Adicionar um sprite para o jogador e vários sprites para os elementos do jogo.
      2. Adicionar lógica para mover o personagem com as teclas de seta.
      3. Adicionar eventos de colisão entre o jogador e os elementos do jogo.
      4. Adicionar pontuação e feedback para o jogador.
  2. Exercício 2: Animação Interativa
    • Criar uma animação onde os personagens interagem através de mensagens.
    • Passo a passo:
      1. Adicionar dois ou mais sprites.
      2. Adicionar lógica para iniciar a animação com um clique ou uma tecla.
      3. Adicionar mensagens para coordenar as ações dos personagens.
      4. Adicionar diálogos e movimentos baseados nos eventos de mensagem.

Materiais Necessários

  • Computadores com acesso à internet e ao site do Scratch.
  • Projetor para demonstração.

Duração Total

2 horas
